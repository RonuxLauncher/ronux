# Ronux

**Ronux** is a **proprietary Roblox launcher for Linux**, written in Rust. It allows Linux users to run Roblox easily without manual Wine or Proton setup, providing a one-click, optimized experience.  

> ⚠ Currently in development – some features may be incomplete or unstable.

---

## Built With

- **Rust** – core launcher logic and system integration  
- **Wine / Proton** – runs Roblox on Linux
